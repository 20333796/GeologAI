🚀 立即可执行的步骤 (5分钟启动)
================================================================================

方式A: Docker启动 (推荐)
  1. cd D:\GeologAI
  2. docker-compose up -d
  3. docker-compose exec backend python -m app.db.init_db
  4. 访问 http://localhost:8000/api/docs

方式B: 本地启动
  1. cd backend
  2. pip install -r requirements.txt
  3. cp .env.example .env
  4. python -m app.db.init_db
  5. uvicorn app.main:app --reload --port 8000
  6. 访问 http://localhost:8000/api/docs
