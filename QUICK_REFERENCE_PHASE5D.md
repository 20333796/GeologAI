# 🌍 GeologAI Phase 5D 快速参考卡

## ⚡ 快速启动 (60秒)

```powershell
# 在项目根目录执行
cd d:\GeologAI
python start_app.py
```

打开浏览器: http://localhost:8501

## 🎯 核心功能

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 注册新账户 | 点击 📝 注册 | 创建新用户 |
| 登录账户 | 点击 🔐 登录 | 使用现有账户 |
| 进入仪表盘 | 登录后自动 | 查看功能模块 |
| 快速操作 | 上方3按钮 | 常用功能 |
| 功能模块 | 下方网格 | 10大功能 |
| 退出登录 | 点击 🚪 退出 | 返回首页 |

```

├─ Logo (左)

模态框
├─ 登录表单
└─ 注册表单
```

## 🐛 常见问题速解

| 问题 | 解决方案 |
|------|---------|
| 页面空白 | Ctrl+Shift+R 刷新 |
| 连接失败 | 检查后端是否运行 |
| 注册失败 | 检查用户名/密码规则 |
| 模态框样式错误 | F5 重新加载 |
| 功能点击无反应 | 检查是否已登录 |

## 🔧 手动启动

**后端:**
```powershell
cd backend
python run_backend.py
```

**前端:**
```powershell
cd web/frontend
streamlit run app.py --server.port 8501
```
