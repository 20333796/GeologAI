### 4️⃣ 认证流程集成

**登录流程**:
```
输入用户名/密码 → 验证 → 获取JWT → 存储Session State → 重新运行 → 显示仪表盘
```

**注册流程**:
```
填写信息 → 创建用户 → 自动登录 → 获取JWT → 显示仪表盘
```

**退出流程**:
```
点击退出 → 清空Session State → 重新运行 → 显示首页
```
