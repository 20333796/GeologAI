## 🚀 快速开始（3步启动）

### 方式1: 自动启动脚本（推荐）

```powershell
# 在项目根目录运行
cd d:\GeologAI
conda activate geologai
python start_app.py
```

这会同时启动后端和前端。

### 方式2: 手动启动

**第一步：启动后端**
```powershell
cd d:\GeologAI\backend
conda activate geologai
python run_backend.py
```

后端将在 `http://127.0.0.1:8001` 运行

**第二步：启动前端**
```powershell
cd d:\GeologAI\web\frontend
conda activate geologai
streamlit run app.py --server.port 8501
```

前端将在 `http://localhost:8501` 打开

## 🐛 故障排除

### 问题1: 打开首页显示空白
**解决方案:**
```powershell
# 清除缓存并刷新
Ctrl + Shift + R  # Windows/Linux
Cmd + Shift + R   # macOS
```

### 问题2: 注册失败，提示400错误
**检查项:**
- 用户名长度: 4-20字符
- 密码长度: 至少8字符
- 邮箱格式: 正确的email
- 用户名不重复

### 问题3: 登录成功但页面未刷新
**解决方案:**
```powershell
# 手动刷新页面
F5 或 Ctrl + R
```

### 问题4: 功能模块显示但点击无反应
**检查项:**
1. 已登录？ (右上角显示用户名)
2. 后端运行？ (http://127.0.0.1:8001/docs)
3. 网络连接正常？

### 问题5: 后端连接失败
**解决方案:**
```powershell
# 验证后端
curl http://127.0.0.1:8001/docs

# 检查端口占用
netstat -ano | findstr :8001

# 重启后端
python backend/run_backend.py
```
