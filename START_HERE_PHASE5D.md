## 🔧 技术栈

### 前端
```
Streamlit 1.51.0
├─ Session State: 状态管理
├─ Markdown: HTML/CSS渲染
├─ Button: 按钮交互
├─ Input: 表单输入
└─ Columns: 布局容器
```

### 后端
```
FastAPI 0.104.1
├─ JWT: 身份认证
├─ SQLAlchemy: 数据库ORM
├─ Uvicorn: ASGI服务器
└─ SQLite: 数据存储
```

### 样式系统
```
CSS3
├─ Grid Layout: 栅栏系统
├─ Flexbox: 弹性布局
├─ Animations: 动画效果
├─ Gradients: 渐变背景
└─ Transforms: 图形变换
```

## 🐛 故障排除速记

| 问题 | 原因 | 解决 |
|------|------|------|
| 页面空白 | 缓存问题 | Ctrl+Shift+R |
| 连接失败 | 后端未启 | 运行后端 |
| 注册失败 | 参数错误 | 检查规则 |
| 无反应 | 未登录 | 登录账户 |
| 样式错误 | CSS冲突 | F5刷新 |
